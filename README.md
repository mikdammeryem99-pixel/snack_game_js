# 🐍 Neon Snake Game

A modern, sleek web-based Snake game built with vanilla JavaScript, HTML, and CSS. 

## ✨ Features
- **Modern UI**: Dark mode aesthetics with smooth glassmorphism effects.
- **Responsive Controls**: Play with keyboard arrows or on-screen buttons for mobile devices.
- **High Score Tracking**: Saves your best score locally so you can beat your own record.
- **CI/CD Pipeline**: Fully automated deployment to GitHub Pages via GitHub Actions.

## 🚀 Play Now
You can play the live version of the game directly from the GitHub Pages link!

## 💻 Tech Stack
- HTML5 (Canvas)
- CSS3
- Vanilla JavaScript (ES6+)
- GitHub Actions (CI/CD)

## 🛠️ Local Setup
1. Clone the repository: 
```bash
git clone https://github.com/mikdammeryem99-pixel/snack_game_js.git
```
2. Open `index.html` in your favorite browser.
3. Enjoy!
